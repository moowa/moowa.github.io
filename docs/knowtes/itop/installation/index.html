ITOP Installation and Upgrade Notes

Installing Dockerized ITOP 2.1‚Äã

  • Get docker image: docker pull vbkunin/itop:2.1.0

  • Run docker container: sudo docker run -d -p 8000:80 -p 3306:3306 --name=itoptest210 vbkunin/itop:2.1.0

  • Get admin password sudo docker exec <containername>/create_mysql_admin_user.sh - will not actually create admin user, but will display admin password?

  • Change admin password inside docker mysql

  • SET PASSWORD FOR 'admin'@'%' = PASSWORD('newpassword')
  • FLUSH PRIVILEGES
  • Manually install toolkit
  • Extract zip to folder named toolkit in topmost directory
  • Launch toolkit by pointing browser to localhost:<port>/toolkit

Installing Dockerized ITOP 2.2 in another container‚Äã

  • container name: upitop220
  • db name: itopdb220
  • Get docker image: docker pull vbkunin/itop:2.2.0
  • Run docker container: sudo docker run -d -p 8000:80 -p 3306:3306 --name=upitop220 vbkunin/itop:2.2.0

Note: Had to change admin password via mysql due to issue: no /app/conf folder. To resolve, do:

  • Copy dump to temporary directory inside the container: docker cp -a source_itop-dump.sql containername:/tmp/
  • Dump db from 2.1: mysql -u___ -p___ dbname < /tmp/itop-dump.sql
  • Copy extensions directory
  • Go to localhost:8000/setup
  • Go through wizard, make sure Add Attribute is selected in extensions (if extension was made in previous version)

Upgrading 2.2 to 2.7‚Äã

  • Install dockerized 2.7.0 sudo docker run -d -p 8000:80 -p 3306:3306 --name=upitop270 vbkunin/itop:2.7.0 Note: go to localhost:8000/setup using firefox, can‚Äôt click the checkbox using chrome
  • Check admin password using sudo docker logs upitop270 | grep -A7 -B1 "Your MySQL user 'admin' has password:"
  • dbname: upitopdb270
  • container: upitop270

  • docker cp -a itop_dump.sql upitop270:/tmp/

  • Dump db from 2.2

  • mysql -u__ -p__ upitopdb270 < /tmp/itop_dump.sql
  • docker cp -a extensions/sample-add-attribute upitop270:/var/www/html/extensions/
  • In docker bash, ./make-itop-config-writable.sh
  • In browser, go to localhost:8000/setup
  • Go through wizard, make sure Add Attribute is selected in extensions (if extension was made in previous version)
